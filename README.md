# Python Palindrome Checker

A lightweight, beginner-friendly Python script that determines whether a given word, phrase, or sentence is a palindrome. This project was built as a hands-on exercise to brush up on Python syntax, string manipulation, and core programming concepts after a hiatus.

## 🚀 Features
* **Case-Insensitivity:** Accurately identifies "Racecar" or "Madam" as palindromes.

## 🛠️ Installation & Setup

### Prerequisites
Make sure you have Python 3.x installed on your machine. You can check your version by running:
```bash
python3 --version
```


## 💻 How to Run
Execute the script directly from your terminal:

```bash
python3 palindrome.py
```
